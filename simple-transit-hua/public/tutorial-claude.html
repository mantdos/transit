Claude Code 安装教程

Windows 安装教程

前置要求: 安装 Git

确保已安装 Git，如果尚未安装，可以点击前往→ Git 官网 下载并安装（安装时一直点击“下一步”即可）。

前置要求：安装 Node.js

确保 Node.js 版本 ≥ 18.0

检查是否已安装 Node.js

node --version

如果版本低于 18.0 或未安装，请点击前往→ Node.js 官网 下载安装最新 LTS 版本。

步骤 1：卸载其他中转客户端（可选）

如果之前安装过其他 Claude Code 客户端，建议先卸载。

检查安装位置

检查是否在本地项目中安装：

npm ls @anthropic-ai/claude-code

检查是否被全局安装：

npm ls -g @anthropic-ai/claude-code

执行卸载操作

卸载本地安装的包：

npm uninstall @anthropic-ai/claude-code

卸载全局安装的包：

npm uninstall -g @anthropic-ai/claude-code

步骤 2：安装 Claude Code

方法 1：使用 PowerShell 脚本安装（推荐）

# 以管理员身份运行 PowerShell
irm https://claude.ai/install.ps1 | iex

方法 2：使用 npm 全局安装

npm install -g @anthropic-ai/claude-code

验证安装

claude --version

步骤 3：配置环境变量

您需要配置两个环境变量：

  • ANTHROPIC_AUTH_TOKEN：您的授权令牌（格式：sk_...）
  • ANTHROPIC_BASE_URL：API 服务地址（本站提供的连接点 URL： __SITE_ORIGIN__）

临时配置（当前会话有效）

方法1：使用 PowerShell

$env:ANTHROPIC_BASE_URL="__SITE_ORIGIN__"
$env:ANTHROPIC_AUTH_TOKEN="您的Token"

方法2：使用命令提示符（CMD）

set ANTHROPIC_BASE_URL=__SITE_ORIGIN__
set ANTHROPIC_AUTH_TOKEN=您的Token

永久配置（推荐）

方法1：通过系统环境变量设置

  1. 右键点击“此电脑”，选择“属性”
  2. 点击“高级系统设置”
  3. 点击“环境变量”
  4. 在“用户变量”区域，点击“新建”，分别添加：
    • 变量名：ANTHROPIC_BASE_URL，变量值：您的连接点 URL
    • 变量名：ANTHROPIC_AUTH_TOKEN，变量值：您的 Token
  5. 点击“确定”保存
  6. 重启终端或命令提示符

方法2：通过 PowerShell 配置

在 PowerShell 中运行以下命令设置用户级环境变量：

[System.Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "__SITE_ORIGIN__", [System.EnvironmentVariableTarget]::User)

[System.Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", "你的API密钥", [System.EnvironmentVariableTarget]::User)

在 PowerShell 中查看已设置的环境变量：

[System.Environment]::GetEnvironmentVariable("ANTHROPIC_BASE_URL", [System.EnvironmentVariableTarget]::User)

[System.Environment]::GetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", [System.EnvironmentVariableTarget]::User)

步骤 4：开始使用

首次运行配置

运行后需要完成以下步骤：

  1. ✅ 安装Git（如尚未安装，可从 Git 官网下载并安装）
  2. ✅ 选择你喜欢的主题 + Enter
  3. ✅ 确认安全须知 + Enter
  4. ✅ 使用默认 Terminal 配置 + Enter
  5. ✅ 信任工作目录 + Enter

提示：如果不是第一次运行，可以跳过以上步骤。

配置好环境变量后，在您的项目目录下使用终端运行：

cd your-project-folder
claude 你是什么大模型

当然，你也可以在 VS Code 或 Cursor 中安装插件 Claude Code for VS Code 来获得更好的使用体验。

完成后，就可以开始在终端里和你的 AI 编程搭档一起写代码了！🚀

常见问题

❓ 提示找不到 claude 命令

解决方法：

  1. 确认已正确安装：npm ls -g @anthropic-ai/claude-code
  2. 重启终端窗口
  3. 检查 npm 全局路径是否在 PATH 中

❓ 环境变量设置后不生效

解决方法：

  • 完全关闭所有终端窗口，重新打开
  • 如果是永久配置，需要重启电脑
  • 使用 echo %ANTHROPIC_AUTH_TOKEN% 验证变量是否设置成功

❓ PowerShell 脚本执行被阻止

解决方法：

# 以管理员身份运行 PowerShell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

macOS 安装教程

前置要求: 安装 Git

确保已安装 Git，如果尚未安装，可以点击前往→ Git 官网 下载并安装。

前置要求：安装 Node.js

确保 Node.js 版本 ≥ 18.0

安装 Xcode Command Line Tools

sudo xcode-select --install

安装 Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装 Node.js

brew install node

验证安装

node --version

步骤 3：配置环境变量

  • ANTHROPIC_BASE_URL：API 服务地址（连接点：__SITE_ORIGIN__）

方法 1：临时配置（当前会话有效）

export ANTHROPIC_AUTH_TOKEN="您的Token"

方法 2：永久配置（推荐）

检查当前使用的 Shell

Zsh 配置（macOS 默认）

echo -e '\nexport ANTHROPIC_AUTH_TOKEN="您的Token"' >> ~/.zshrc
echo -e '\nexport ANTHROPIC_BASE_URL="__SITE_ORIGIN__"' >> ~/.zshrc

source ~/.zshrc

Bash 配置

echo -e '\nexport ANTHROPIC_AUTH_TOKEN="您的Token"' >> ~/.bash_profile
echo -e '\nexport ANTHROPIC_BASE_URL="__SITE_ORIGIN__"' >> ~/.bash_profile
echo -e '\nexport ANTHROPIC_AUTH_TOKEN="您的Token"' >> ~/.bashrc
echo -e '\nexport ANTHROPIC_BASE_URL="__SITE_ORIGIN__"' >> ~/.bashrc

source ~/.bash_profile
source ~/.bashrc

Linux 安装教程

前置要求: 安装 Git

确保已安装 Git（如果尚未安装，请根据你的发行版安装）。

前置要求：安装 Node.js

确保 Node.js 版本 ≥ 18.0

Ubuntu / Debian 系列

# 添加 NodeSource 仓库并安装 Node.js LTS
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo bash -
sudo apt-get install -y nodejs

# 验证安装
node --version

Fedora / RHEL / CentOS 系列

# 安装 Node.js
sudo dnf install -y nodejs

# 验证安装
node --version

Arch Linux / Manjaro

sudo pacman -S nodejs npm
node --version
