快速生成 Claude Code 环境配置

Claude Code 配置文件

通过 cc-switch(点击查看下载与使用指南) ，您可以为Claude Code添加自定义的模型提供商。请参考以下 JSON 配置，将其中的 ANTHROPIC_AUTH_TOKEN 替换为您在控制台获取的专属 Token。 

{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "sk-您的专属Key",
    "ANTHROPIC_BASE_URL": "https://codex.jc-space.cloud",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "gpt-5.4",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "gpt-5.4",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "gpt-5.4",
    "ANTHROPIC_MODEL": "gpt-5.4",
    "ANTHROPIC_REASONING_MODEL": "gpt-5.4"
  },
  "includeCoAuthoredBy": false
}
